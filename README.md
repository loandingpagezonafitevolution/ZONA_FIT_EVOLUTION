 Zona Fit Evolution - Landing Page

##  Descripción

Landing Page desarrollada para **Zona Fit Evolution**, un sistema de gestión administrativa para gimnasios. El objetivo principal es presentar de manera atractiva las funcionalidades del sistemay  sus beneficios

---

##  Objetivo

Promocionar la plataforma de administración para gimnasios, destacando:

* Gestión de clientes.
* Control de membresías.
* Registro de pagos.
* Control de acceso.
* Generación de reportes.
* Seguridad de la información.

---

##  Secciones de la Landing Page

### 1. Hero Principal

Presentación del sistema con llamada a la acción.

### 2. Funcionalidades

Muestra los módulos principales:

* Gestión de Clientes
* Membresías
* Pagos
* Control de Acceso
* Gestión de Empleados
* Reportes

### 3. Beneficios

Ventajas que ofrece el sistema para la administración del gimnasio.

### 4. Estadísticas

Indicadores visuales sobre clientes, membresías, pagos y accesos.

### 5. Contacto

Formulario para solicitar información o demostraciones.

### 6. Preguntas Frecuentes

Respuestas rápidas a dudas comunes de los usuarios.

### 7. Footer

Información de contacto, enlaces rápidos y redes sociales.
